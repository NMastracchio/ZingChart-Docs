.. glossary::
   :sorted:

   text
     *String.* Sets the displayed text of the object.

     **Default:** null
 
   text-align
     *String.* Sets the horizontal alignment of the text.
 
     **Accepted values:** "left" | "center" | "right"
 
     **Default:** "center"
 
   vertical-align
     *String.* Sets the vertical alignment of the text.
 
     **Accepted values:** "top" | "middle" | "bottom"
 
     **Default:** "middle"
 
   font-size
     *Integer.* Sets the size of the font in pixels.
 
     **Default:** 11
 
   font-family
     *String.* Sets the font family.
 
     **Default:** "Lucida Sans Unicode,Lucida Grande,Verdana,Arial,sans-serif"
 
   color
     *String.* Sets the color of the font.
 
     **Default:** "#000"
 
   bold
     *Boolean.* Sets whether or not the text will be bolded.

     **Default:** false
 
   italic
     *Boolean.* Sets whether or not the text will be italicized.

     **Default:** false
 
   underline
     *Boolean.* Sets whether or not the text will be underlined.

     **Default:** false
 
   max-chars
     *Integer.* Sets the maximum number of characters to display, dependent on the width of the containing object.

     **Default:** 0
 
   padding-top
     *Integer.* Sets the top padding of the object in number of pixels.

     **Default:** 2
 
   padding-right
     *Integer.* Sets the right padding of the object in number of pixels.

     **Default:** 2
 
   padding-bottom
     *Integer.* Sets the bottom padding of the object in number of pixels.

     **Default:** 2
 
   padding-left
     *Integer.* Sets the left padding of the object in number of pixels.

     **Default:** 2
 
   clip-text
     *Boolean.* Sets whether or not the text will clip when text extends beyond the width of the containing object.

     **Default:** false
 
   wrap-text
     *Boolean.* Sets whether or not the text will wrap when text extends beyond the width of the containing object.

     **Default:** false
 
   line-height
     *Integer.* Sets the line-height.

     **Default:** -1
 
   text-height
     *Integer.* Sets the text-height.

     **Default:** 0
 
   text-width
     *Integer.* Sets the width of the text.

     **Default:** 0
 
   max-width
     *Integer.* Sets the maximum width of the element.

     **Default:** ZC.MAX
 
   rtl
     *Boolean.* Sets whether or not the text is written from right-to-left.

     **Default:** false
