Scale-X¶

Elements¶

guide
Object. Configures the guide element.
item
Object. Styling options for the item labels at each axis.
markers
Array<Object>. Set and style marker objects to call out information.
minor-guide
Object. Configures the minor guide line elements that appear between major guide lines.
minor-tick
Object. Configures the minor tick elements that appear between the primary tick elements on a scale.
ref-line
Object. Configures the reference line.
tick
Object. Configure the tick elements.
transform
Object. Sets the Unix time transform options.
label
Object. Sets the scale’s label.
