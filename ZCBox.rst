.. glossary::
   :sorted:

   shape 
     *String.* The type of shape.

     **Default:** "box"
 
   width 
     *Integer.* The width of the shape.

     **Default:** 0

   height 
     *Integer.* The height of the shape.

     **Default:** 0

   position 
     *String.* The position of the shape.

     **Default:** ""

   margin-top 
     *Integer.* The top margin of the shape.

     **Default:** -1

   margin-right The right margin of the shape.
     *Integer.* 

     **Default:** -1

   margin-bottom The bottom margin of the shape.
     *Integer.* 

     **Default:** -1

   margin-left The left margin of the shape.
     *Integer.* 

     **Default:** -1

   border-radius-top-left 
     *Integer.* The top left border radius of the shape.

     **Default:** 0

   border-radius-top-right 
     *Integer.* The top right border radius of the shape.

     **Default:** 0

   border-radius-bottom-right 
     *Integer.* The bottom right border radius of the shape.

     **Default:** 0

   border-radius-bottom-left 
     *Integer.* The bottom left border radius of the shape.

     **Default:** 0

   center-ref 
     *Boolean.* When true, the object reference point will be the center of the shape.

     **Default:** false

   callout 
     *Boolean.* Sets whether or not a callout will be displayed.

     **Default:** false

   callout-position 
     *String.* Sets the position of the callout.

     **Accepted values:** "top" | "bottom" | "left" | "right"

     **Default:** "bottom"

   callout-offset 
     *Integer.* Sets the offset of the callout in number of pixels.

     **Default:** 0

   callout-extension 
     *Integer.* Sets the length of the callout extension.

     **Default:** 0

   callout-width 
     *Integer.* Sets the width of the callout.

     **Default:** 8

   callout-height 
     *Integer.* Sets the height of the callout.

     **Default:** 8

   callout-ratio 
     *array.* Sets the ratio of the callout?

     **Default:** [1,1]

   callout-hook 
     *array.* Sets the hook location of the callout.

     **Default:** null

   callout-tip 
     *object.* Sets the styling attributes for the tip of the callout.

     **Default:** null

   ignore-locate 
     *Boolean.* Ignores and locates? No idea.

     **Default:** false

   box-correction 
     *Boolean.* Corrects boxes.

     **Default:** true

   border-top 
     *String.* Sets the styling attributes of the top border.

     **Default:** ""

   border-right 
     *String.* Sets the styling attributes of the right border.

     **Default:** ""

   border-bottom 
     *String.* Sets the styling attributes of the bottom border.

     **Default:** ""

   border-left 
     *String.* Sets the styling attributes of the left border.

     **Default:** ""

   anchor 
     *String.* Sets the anchor position of the element.

     **Default:** "tl"

   rect-shortcut 
     *Boolean.* Does something that idk.

     **Default:** true
